# Code of Conduct

## Our Standards

**Welcome:** inclusive language, respect for differing views, constructive criticism, empathy.

**Not welcome:** harassment, discrimination, offensive comments, unwanted contact, publishing private info, spam.

## Scope

Applies to all project spaces: GitHub issues, discussions, pull requests, and any public space where someone represents this project.

## Enforcement

Report violations privately to the project maintainer (see GitHub profile). All reports are reviewed confidentially. Actions range from warnings to permanent removal.
